# Manual de Usuario RMC Inventory

Audiencia: usuarios administrativos y operativos que registran inventario, compras, proyectos, recepción, despachos y reportes.

Objetivo: presentar el proyecto como un manual práctico, con capturas reales y pasos de operación claros.

Narrativa:
1. Presentar el sistema y su alcance operativo.
2. Explicar la navegación general.
3. Mostrar los módulos principales con capturas.
4. Resumir el flujo recomendado de uso.
5. Cerrar con buenas prácticas operativas.

Slides:
1. Portada del manual.
2. Alcance del sistema.
3. Navegación principal.
4. Dashboard.
5. Productos.
6. Almacenes.
7. Proyectos.
8. Compras y recepción.
9. Reportes.
10. Configuración.
11. Flujo recomendado.
12. Buenas prácticas.

Fuente de contenido:
- Código del proyecto Next.js.
- Traducciones y nombres de módulos en `src/locales/es.json`.
- Capturas reales tomadas desde `http://localhost:3000`.

Sistema visual:
- Manual corporativo limpio, fondo claro, acentos verdes/teal y negro.
- Tipografías sans modernas.
- Capturas en marcos amplios con llamadas numeradas.
- Texto editable en PowerPoint para títulos, instrucciones y notas.

Editabilidad:
- Todos los títulos, pasos, etiquetas y notas deben ser objetos editables.
- Las capturas se insertan como imágenes embebidas.
